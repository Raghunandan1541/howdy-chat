# howdy-chat

### Create .env file
 
###### PUSHER_APP_ID= id
###### PUSHER_KEY= key
###### PUSHER_SECRET=secret
###### PUSHER_CLUSTER=eu

###### MONGO_URL = URL

###### PORT = 4000

### Installation

1. Use `npm install` in root directory.
2. Use `cd client` and the use command `npm install` to install react app.

### Running

Use `npm run dev` in root directory to start the app.
